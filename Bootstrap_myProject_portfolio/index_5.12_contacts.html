Однако, с помощью фреймворка Bootstrap можно создать специальную форму для обратной связи.


Для добавления поля для ввода сообщения в форму обратной связи в Bootstrap, нужно использовать элемент textarea.

Тег textarea создает многострочное поле ввода.
Атрибут class="form-control" применяет стили Bootstrap для поля ввода.
Атрибут id="message" задает уникальный идентификатор для поля ввода.
Атрибут rows="3" определяет количество строк в поле.
Атрибут required указывает, что поле обязательно для заполнения.

Дальше для создания кнопки отправки используем тег button с атрибутом type="submit", который указывает, что это кнопка отправки формы. Классы btn и btn-primary задают стили кнопки.

После того как мы создали форму обратной связи с помощью Bootstrap, нам нужно добавить обработчик формы, который будет обрабатывать введенные данные и отправлять их на сервер.

Для начала, создадим JavaScript-функцию, которая будет выполняться при отправке формы. Мы добавим эту функцию внутри тега script после кода, который инициализирует Bootstrap.
