Мой проект №3

"Список дел он-лайн."

Скрин-шот


Данный проект был разработан мною согласно Практическому заданию № 7: разработать «Приложение Список дел».
В итоге изучены основы Vue.js и создано простое приложение для составления списка дел.

Рассмотрены понятия:
- директива v-for для вывода списка дел из массива
- компонент формы для добавления новых пунктов в список дел
- директива v-model
- директива v-on
- кнопка для удаления элементов из списка
- обработка события нажатия на кнопку и удаление элемента из списка
- добавление функций сортировки и фильтрации списка

Изучить основы Vue.js и создать простое приложение для составления списка дел. 1. Создайте новый проект Vue.js. 2. Подключитесь и получите данные с помощью открытого API к https://jsonplaceholder.typicode.com/ (или к любому свободному API) 3. Создайте новый компонент для отображения списка дел. 4. Используйте директиву v-for для вывода списка дел из массива. 5. Добавьте компонент формы для добавления новых пунктов в список дел.
