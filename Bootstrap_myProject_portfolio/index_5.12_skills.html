HTML


HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.(HTML)

Изучены теги: head, body, h-заголовок, p-абзац, br-пробел, ul li - списки, div, img, a href - ссылки, nav-навигация, меню, style-элементы оформления стиля и другие


CSS

CSS (от англ. Cascading Style Sheets «каскадные таблицы стилей») — формальный язык декорирования и описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML). (CSS)

Изучены темы: class, id, hover, параметры: margin, paddin, border, background,display, box-shadow, width, height и другие

JavaScript

JavaScript — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Является реализацией спецификации ECMAScript (стандарт ECMA-262). (JavaScript)

Для добавления JavaScript-кода на страницу можно использовать теги script, которые рекомендуется, но не обязательно, помещать внутри контейнера head или создать отдельный файл с расширением js.

Изучены: переменные, константы, типы данных, число. строка, булевый тип, взаимодействие: alert, prompt, confirm, вывод console log, условные операторы if, логические И ИЛИ НЕ, циклы for и while и другое. Многое из вышеперечисленного было применено при разработке моего проекта "Калькулятор он-лайн".
